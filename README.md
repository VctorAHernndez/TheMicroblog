# The Microblog
My first Flask app ever.

### Description
It consists of a MySQL database and a Flask server, with pages rendered with Jinja.

### API Endpoints

##### Home Page
`GET /`

##### About Page
`GET /about`

##### Articles Page
`GET /articles`

##### Article Page (Singular)
`GET /article/:article_id`

##### Register Page
`GET:POST /register`

##### Login Page
`GET:POST /login`

##### Logout
`GET /logout`

##### Dashboard Page
`GET /dashboard`

##### Add Article Page
`GET:POST /add_article`

##### Edit Article Page
`GET:POST /edit_article/:article_id`

##### Delete Article
`POST /delete_article/:article_id`
